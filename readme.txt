
This application will take any number b/w 0 to 10000000 and o/p the magic number, which will have different digits in it.
All the digits in o/p number will be different from what user used in i/p number. It will give the o/p with same lenght 
as of i/p number.

Steps to run this application:

	1) Download the zip
	2) run "npm install" // it will install all the dependencies used by application
	3) run "node ./bin/www" // it will start the node server to get the request 
	4) Open in browser "http://localhost:3000"

** Note : If you want to change any functionality. please go to index.html file in views folder.
